Defense Layer

Six layers of protection

Every email passes through deterministic, regex-based analysis before your agent sees a single token. No AI in the loop—pure, auditable rules.

⚠

Injection Detection

Catches prompt injection, jailbreak attempts, role-override patterns, and instruction hijacking with 40+ detection rules.

✖

HTML & Script Stripping

Removes all HTML, CSS, JavaScript, hidden elements, tracking pixels, and invisible text. Your agent sees only clean plaintext.

●

Secret Redaction

Detects and redacts API keys, passwords, JWTs, AWS credentials, and other secrets before they leak into agent context.

⚙

Attachment Filtering

Allowlist-based attachment policy. Blocks executable payloads, extracts safe text, reports blocked files with metadata.

★

Risk Scoring

Every email receives a 0–100 risk score with granular flags: injection, HTML, secrets, Unicode abuse, hidden content.

⚡

Real-Time Monitoring

Live dashboard with event stream, timeline graphs, sender analytics, and per-inbox filtering. See threats as they arrive.
